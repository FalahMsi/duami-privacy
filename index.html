Privacy Policy – Duami

Last updated: January 20, 2026

1. Data Collection

Duami does not collect, store, or transmit any personal data outside the user’s device. All information entered by the user (such as shift schedules or leave records) is stored locally on the device only.

2. Use of Data

Data is used only to display work shift schedules, manage leave records, and generate reports (PDF / Excel).

3. Data Sharing

Duami does not share any data with third-party services, external servers, advertising networks, or any other external entities.

4. Notifications

The application may use local notifications to remind users of upcoming work shifts. No data is transmitted over the internet for notification purposes.

5. Calendar Access

If calendar integration is enabled, Duami accesses the device’s local calendar only to read official holidays. The application does not modify or transmit calendar data outside the device.

6. Data Security

All data is stored locally on the device using standard iOS security mechanisms.

7. Independent Application

Duami is an independent application and is not affiliated with or officially associated with any government entity.

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be published within the application.

9. Contact

If you have any questions regarding this Privacy Policy, you may contact us at:
