Privacy Policy

Last updated: January 20, 2026

1. Data Collection

Duami does not collect, store, or transmit any personal data outside the user’s device. All information entered by the user is stored locally on the device only.

2. Use of Data

Data is used only to display work shift schedules, manage leave records, and generate reports (PDF / Excel).

3. Data Sharing

Duami does not share any data with third parties, external servers, advertising services, or analytics platforms.

4. Notifications

The application may use local notifications to remind users of upcoming work shifts. No data is transmitted over the internet.

5. Calendar Access

If enabled, Duami accesses the device calendar only to display official holidays. No calendar data is modified or transmitted.

6. Data Security

All data is stored locally using standard iOS security mechanisms.

7. Independent Application

Duami is an independent application and is not affiliated with any government entity.

8. Policy Changes

This Privacy Policy may be updated from time to time. Any changes will be published within the application.
